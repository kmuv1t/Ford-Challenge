Nossa Solução

O objetivo do projeto é eliminar um problema que afeta a maioria dos motoristas de carros modernos, a distração. Com o avanço da tecnologia, obtivemos diversas ferramentas para auxiliar nosso dia a dia, porém, infelizmente muitas delas, em certos momentos, mais atrapalhar do que ajudar. O exemplo disso são os smartphones e multimídias inteligentes, que podem distrair facilmente o motorista podendo causar eventuais acidentes graves. Podemos exemplificar também o quanto o painel do carro “incentiva” o uso constante dele.


O hardware planejado se trata de Raspberry Pi 0 como microcomputador, um microfone (USB), display sensível ao toque de (3,5”), ventoinha (50mm) para refrigeração, cabo OTG (micro USB), alto falante. Estamos cogitando um Arduino para integração entre computador e a multimídia para utilizar de um simulador de direção para demonstrar nosso projeto.


Ambiente de desenvolvimento, refinamento dos comandos do Google Assistant para melhor a integração com o carro e com o trânsito. Estamos utilizando Python como linguagem principal, tanto nossa GUI quanto o código de execução do Assistant estão rodando na mesma linguagem.


Até o momento desenvolvemos: Google Assistant, a interface gráfica e a integração de ambos, desenvolvidos pelo Figma, traduzido em Python, impressão da case em 3D, e um shell script para integração.
